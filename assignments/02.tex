\section{Truth Tables}\label{truth-tables}}

Using \url{http://carnap.io}, build truth tables for the following
sentences.

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \(P \rightarrow P\)
\item
  \(Q \rightarrow \neg Q\)
\item
  \((P \rightarrow R) \vee (R \rightarrow P)\)
\item
  \((P \wedge R) \rightarrow (R \rightarrow P)\)
\item
  \(\neg ((P \wedge Q) \vee (P \vee Q))\)
\item
  \((Q \rightarrow (R \vee P)) \wedge \neg Q\)
\item
  \((P \wedge \neg P) \rightarrow R\)
\item
  \(\neg \neg (\neg P \wedge \neg R)\)
\item
  \(((P \wedge R) \rightarrow Q) \rightarrow \neg (P \vee R)\)
\item
  \(((P \rightarrow R) \rightarrow Q) \rightarrow (P \rightarrow (R \rightarrow Q))\)
\end{enumerate}
